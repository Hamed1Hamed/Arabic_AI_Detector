pip install -r requirements.txt
You only need to modify the config.json file in order to change the hyperparameters of the model.
We used the following models for the experiments:
    "model_name": "aubmindlab/araelectra-base-discriminator",
    "model_name": "XML-roberta-base",

If you are using this code on another Arabic or non-Arabic datasets, please visit the ArabicTextDataset.py and remove the dediacritization layer, because this layer
is meant for evaluating the AIRABIC benchmark dataset, and it is not needed for other datasets.
